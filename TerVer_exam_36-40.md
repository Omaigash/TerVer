### 36. Применение закона больших чисел к испытаниям Бернулли.

*   **Теорема Бернулли (Простейшая форма Закона больших чисел):**
    Если в $n$ независимых испытаниях вероятность появления события $A$ постоянна и равна $p$, то при неограниченном увеличении числа испытаний ($n \to \infty$) **относительная частота** появления этого события $\frac{m}{n}$ (где $m$ — число успехов) сходится по вероятности к его теоретической вероятности $p$.
*   **Математическая запись:** Для любого сколь угодно малого $\varepsilon > 0$:
    $\lim_{n \to \infty} P\left( \left| \frac{m}{n} - p \right| < \varepsilon \right) = 1$
*   **Смысл:** Это важнейшая теорема, которая обосновывает **статистическое определение вероятности**. Она объясняет, почему при многократном подбрасывании монеты доля выпавших "орлов" стабилизируется около 0.5. Чем больше испытаний мы проводим, тем ближе эмпирическая частота к истинной вероятности, и шансы на их сильное расхождение стремятся к нулю.

### 37. Центральная предельная теорема (ЦПТ). Ее суть и формула для $S_n$.

*   **Суть ЦПТ:** Если случайная величина является суммой очень большого числа независимых случайных величин, влияние каждой из которых на всю сумму ничтожно мало, то распределение этой суммы **будет близко к нормальному распределению**, *независимо от того, по каким законам были распределены исходные слагаемые*.
    *(Именно поэтому нормальный закон так часто встречается в природе: рост человека — это сумма влияния тысяч различных генов и факторов среды; погрешность измерения — сумма множества мелких помех).*
*   **Формальная постановка для одинаково распределенных величин:**
    Пусть $X_1, X_2, \dots, X_n$ — независимые случайные величины с одинаковым распределением. Пусть их матожидание $M(X_i) = a$, а дисперсия $D(X_i) = \sigma^2$.
    Рассмотрим их сумму: $S_n = X_1 + X_2 + \dots + X_n$.
    По свойствам матожидания и дисперсии: $M(S_n) = n \cdot a$, а $D(S_n) = n \cdot \sigma^2$.
*   **Приближенная формула для суммы $S_n$:**
    Согласно ЦПТ, при достаточно большом $n$ сумма $S_n$ распределена практически нормально. Вероятность того, что сумма попадет в интервал $[x_1, x_2)$, вычисляется через функцию Лапласа $\Phi(x)$:
    $P(x_1 \le S_n < x_2) \approx \Phi\left( \frac{x_2 - na}{\sigma\sqrt{n}} \right) - \Phi\left( \frac{x_1 - na}{\sigma\sqrt{n}} \right)$
    *(Замечание: Интегральная теорема Муавра-Лапласа из билета №15 — это просто частный случай ЦПТ для испытаний Бернулли).*

### 38. Случайная выборка. Вариационный и статистические ряды. Пример.

*   **Понятие случайной выборки:** В статистике мы изучаем генеральную совокупность (всех людей, все детали на заводе и т.д.). **Выборка** — это часть генеральной совокупности, случайно отобранная для изучения. Математически выборка объема $n$ — это набор независимых одинаково распределенных случайных величин $X_1, X_2, \dots, X_n$.
*   **Реализация выборки:** Это конкретные числа (результаты измерений), которые мы получили в ходе эксперимента: $x_1, x_2, \dots, x_n$.
*   **Вариационный ряд:** Это реализация выборки, элементы которой упорядочены (отсортированы) **по возрастанию**: $x_{(1)} \le x_{(2)} \le \dots \le x_{(n)}$.
*   **Сгруппированный (дискретный) статистический ряд:** Если в выборке некоторые значения повторяются, их удобно сгруппировать. Это таблица, где в первой строке записаны уникальные значения выборки по возрастанию ($x_i$), а во второй — их **частоты** $n_i$ (сколько раз значение встретилось). Сумма частот равна объему выборки $n$.
*   **Интервальный статистический ряд:** Применяется для непрерывных величин или когда значений слишком много. Весь диапазон значений выборки разбивают на интервалы (обычно равной длины), и подсчитывают, сколько элементов выборки $n_i$ попало в каждый интервал.
*   **Пример:**
    *   *Реализация выборки (оценки 10 студентов):* 4, 3, 5, 4, 3, 4, 2, 5, 4, 3.
    *   *Вариационный ряд:* 2, 3, 3, 3, 4, 4, 4, 4, 5, 5.
    *   *Сгруппированный статистический ряд:*
        | Значение $x_i$ | 2 | 3 | 4 | 5 |
        | :--- | :--- | :--- | :--- | :--- |
        | Частота $n_i$ | 1 | 3 | 4 | 2 |
        *(Проверка: 1+3+4+2 = 10).*

### 39. Эмпирическая функция распределения. Гистограмма.

*   **Эмпирическая (выборочная) функция распределения $F^*(x)$:** Это статистический аналог теоретической функции распределения $F(x)$. Она показывает долю (относительную частоту) элементов выборки, которые **строго меньше** заданного числа $x$.
    $F^*(x) = \frac{n_x}{n}$
    где $n_x$ — количество элементов выборки, меньших $x$; $n$ — общий объем выборки.
*   **Пример построения (по данным из билета №38):**
    Объем $n=10$. Значения: 2, 3, 4, 5.
    *   Для $x \le 2$: чисел меньше 2 нет ($n_x = 0$) $\Rightarrow F^*(x) = 0/10 = 0$.
    *   Для $2 < x \le 3$: меньше 3 только одно число "2" ($n_x = 1$) $\Rightarrow F^*(x) = 1/10 = 0.1$.
    *   Для $3 < x \le 4$: меньше 4 числа "2" (1 шт) и "3" (3 шт) ($n_x = 4$) $\Rightarrow F^*(x) = 4/10 = 0.4$.
    *   Для $4 < x \le 5$: $n_x = 1+3+4 = 8$ $\Rightarrow F^*(x) = 8/10 = 0.8$.
    *   Для $x > 5$: все числа строго меньше $x$ ($n_x = 10$) $\Rightarrow F^*(x) = 10/10 = 1$.
    График выглядит как ступенчатая лестница со скачками в точках 2, 3, 4, 5.
*   **Свойства $F^*(x)$:**
    1.  $0 \le F^*(x) \le 1$.
    2.  $F^*(x)$ — неубывающая функция.
    3.  Согласно теореме Гливенко-Кантелли, при $n \to \infty$ эмпирическая функция $F^*(x)$ сходится по вероятности к истинной теоретической функции $F(x)$ (выборка хорошо моделирует генеральную совокупность).
*   **Гистограмма:** Это графическое изображение **интервального** статистического ряда (эмпирический аналог графика плотности $f(x)$). Представляет собой ступенчатую фигуру из прямоугольников. Основание прямоугольника — это интервал группировки, а высота равна либо частоте (гистограмма частот), либо плотности частоты $n_i / (n \cdot h)$, где $h$ — ширина интервала (тогда общая площадь гистограммы равна 1).

### 40. Задача оценивания. Виды (свойства) точечных оценок. Выборочное среднее.

*   **Постановка задачи оценивания:** Пусть мы знаем вид распределения (например, нормальное), но не знаем его параметры (например, $a$ и $\sigma$). Наша задача — по результатам случайной выборки $(X_1, \dots, X_n)$ найти такие функции от выборки (они называются **оценками**), которые дадут максимально точные и надежные значения неизвестных теоретических параметров.
*   **Виды точечных оценок (точнее, требования/свойства, которым они должны удовлетворять):**
    Точечная оценка $\hat{\theta}$ (называется "точечной", т.к. дает одно конкретное число-точку) считается хорошей, если она обладает тремя свойствами:
    1.  **Несмещенность:** Математическое ожидание оценки равно истинному параметру: $M(\hat{\theta}) = \theta$. (Оценка не дает систематической ошибки в большую или меньшую сторону).
    2.  **Состоятельность:** При увеличении объема выборки $n \to \infty$ оценка сходится по вероятности к истинному параметру: $\hat{\theta} \xrightarrow{P} \theta$.
    3.  **Эффективность:** Среди всех возможных несмещенных оценок данная оценка имеет самую маленькую дисперсию $D(\hat{\theta})$ (то есть наименьший разброс).
*   **Выборочное среднее ($\bar{X}$):**
    Это точечная оценка теоретического математического ожидания $M(X)$ генеральной совокупности.
    *   **Определение:** Выборочное среднее — это среднее арифметическое всех значений выборки.
        $\bar{X} = \frac{1}{n} \sum_{i=1}^n X_i = \frac{X_1 + X_2 + \dots + X_n}{n}$
    *   **Пример:** Выборка: 2, 4, 9. Выборочное среднее: $\bar{x} = (2 + 4 + 9) / 3 = 15 / 3 = 5$.
    *   **Свойства выборочного среднего:**
        1.  Она является **несмещенной**: $M(\bar{X}) = M(X)$.
        2.  Она является **состоятельной** (это следует напрямую из Закона больших чисел — теоремы Хинчина).
        3.  Дисперсия выборочного среднего равна $D(\bar{X}) = \frac{D(X)}{n}$. (Чем больше выборка, тем точнее среднее).
